Linked To My e_commerce Website

This project is a feature-rich e-commerce website developed using Django. It includes an integrated PayPal payment method, which requires a VPN for access due to regional restrictions in Lebanon. Key features of the project:

  • Cart Management: Users can add, update, and remove items from their cart seamlessly.
  • Email Verification: Ensures secure account registration by verifying users' email addresses
  • Password Reset Functionality: Provides users with the ability to reset their passwords via email.
  • Secure Payment Gateway: The PayPal integration allows for safe and reliable transactions
This project demonstrates my skills in backend development with Django, implementing secure authentication workflows, and integrating third-party APIs for payment processing.
